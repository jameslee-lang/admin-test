Shoplive 어드민 워크스루

실제 Shoplive 어드민 화면 위에 단계별 안내를 띄워, 고객사가 직접 클릭하며 방송을 만들어 보게 하는 도구입니다. 설치가 필요 없습니다. 아래 링크를 북마크바에 드래그해서 저장하고, 어드민 화면에서 누르면 됩니다.

1. 실행할 플로우 선택

선택한 플로우가 아래 "워크스루 실행" 북마클릿에 반영됩니다.

2. 북마클릿 저장 (드래그)

워크스루 실행 요소 선택기

위 버튼을 브라우저 북마크바로 드래그하세요. (클릭하면 이 페이지에서는 동작하지 않습니다 — 실제 어드민 화면에서 눌러야 합니다.)

3. 사용 순서

  1. 이 레포(jameslee-lang/admin-test)에 push 되어 있어야 북마클릿이 정상 동작합니다.
  2. GitHub Pages를 이 폴더로 켜거나, 그냥 이 index.html을 로컬에서 열어도 됩니다.
  3. 위 두 링크(워크스루 실행, 요소 선택기)를 북마크바에 드래그해 저장합니다.
  4. 실제 Shoplive 어드민 페이지로 이동한 뒤 워크스루 실행 북마클릿을 누릅니다.

선택자를 아직 안 모았다면

flows/broadcast-register.json의 선택자는 아직 자리표시자(TODO_SELECTOR_*)입니다. 실제 어드민에서 요소 선택기 북마클릿을 눌러 각 요소를 클릭하면 CSS 선택자가 잡히고, 그 값을 플로우 JSON에 붙여 넣으면 됩니다. 자세한 방법은 README.md를 참고하세요.
